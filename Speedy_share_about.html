In our day to day life we all use multiple devices and all have some files which are needed to be shared in each of the device. Here's the platfrom to avoid all these miss convinience, this platfrom allows file sharing from multiple device like Android, IOS, Mac OS, Windows etc. to and for each.

This platform allows file sharing from multiple device without having any file size limitation, no minimum or maximum distance, by encripting all the data.
