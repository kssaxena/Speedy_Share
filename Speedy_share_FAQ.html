This platfrom does not have any limitations on any size of file. There is no minimum or maximum file size limitaions.

There is no Restriction on file type. This platform supports transfer of every file type wether it is Image, Video, Audio, ZIP, RAR etc.

How to upload Files ?

Can I add password to the uploading file ?

No, there's no option to add password to the file getting uploaded. Also there's no need to add passwords because the platform encrypts the files and sends it.

Can I set expiry dates for shared links ?

No, there is no requirement to set expiry dates for shared links, because the link get expires immideately after sharing is complete.

How can I download files shared with me ?

The shared file once get uploaded will display a link or QR to the first user, which will be transfered to the second user to download the file.
